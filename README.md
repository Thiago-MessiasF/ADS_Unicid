# Repositório Acadêmico – Análise e Desenvolvimento de Sistemas

Este repositório foi criado com o objetivo de armazenar e organizar projetos, atividades e trabalhos realizados ao longo do curso de **Análise e Desenvolvimento de Sistemas**.

## Objetivo

Reunir exercícios, projetos práticos e estudos desenvolvidos durante a graduação, permitindo acompanhar a evolução do aprendizado e manter um histórico organizado das atividades.

## Estrutura do Repositório

Os conteúdos estão organizados por disciplinas, projetos ou tecnologias estudadas ao longo do curso.

Exemplo de organização:

```
/Algoritmos
/Programacao
/Banco_de_Dados
/Desenvolvimento_Web
/Projetos
```

Cada pasta pode conter:

* Códigos-fonte
* Documentação
* Exercícios
* Projetos práticos

## Tecnologias Utilizadas

Durante o curso, diversas tecnologias podem ser utilizadas, como:

* C
* JavaScript
* HTML
* CSS
* Git e GitHub
* Banco de Dados

## Sobre

Este repositório é mantido para fins **acadêmicos e de aprendizado**, reunindo atividades desenvolvidas durante a graduação em **Análise e Desenvolvimento de Sistemas**.
